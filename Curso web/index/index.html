  1. Clasificación de árboles en un parque:
    1. Árboles de hoja caduca: Árboles que pierden sus hojas en otoño.
    2. Árboles perennes: Árboles que mantienen sus hojas durante todo el año.
    3. Árboles frutales: Árboles que producen frutas comestibles, como manzanas o peras.
    4. Árboles coníferos: Árboles que producen conos, como pinos y abetos.
  2. Observa el tamaño y la forma de los árboles para identificar sus especies.
  3. Revisa la corteza y las hojas para obtener más pistas sobre el tipo de árbol.
  4. Busca la presencia de frutos o flores para facilitar la identificación.
